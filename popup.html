How to use this plugin

This plugin allows you to query the DMMapp database without going to the DMMapp. Simply start typing in the search box.

You can search by:

    Library ("Getty", "Laurenziana", etc.)
    Country ("Italy", "France", etc.)
    City ("Rome", "Oxford", etc.)
